# Jogo da Forca

## Autor
Ray Silva Matos

## Descrição
Este projeto é um jogo da forca desenvolvido em JavaScript utilizando Node.js e executado no terminal.

## Regras do jogo
O jogador deve descobrir a palavra secreta antes de atingir o limite máximo de erros.

## Como jogar
1. Digite seu nome
2. Escolha uma categoria
3. Digite letras para tentar descobrir a palavra
4. Vença antes da forca completar

## Funcionalidades
- Sistema de categorias
- Dicas
- Pontuação
- Forca em ASCII
- Jogar novamente

## Como executar
Instale as dependências:
npm install
Execute o jogo:
npm start

## Tecnologias utilizadas
- JavaScript
- Node.js
- prompt-sync
